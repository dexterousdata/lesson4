Structuring a HTML Document

Avoid Repetition

To be a good programmer it is key to avoid repetition. This is done in CSS by applying style to groups of objects.

...The Structure Metaphor

Using a house for the metaphor, HTML is the floor plan of a house. It is the structure.
CSS is the interior design - colours, style, the overall feel
Java Script is the interactive parts - TV remote, garage door remote etc.
DOM is the address of any specific house or (box) - a tree structure of country, state, suburb, street and finally house number is like the location of a section of a page.

The Tree Structure

Grids, Boxes and Rules

A webpage is broken into a grid which all elements are boxes and rectangles which are arranged in the grid.
Suggested to start with the largest boxes working down to the smallest.
Label div's with a class name that describes what the div is - aids in code readability
