Object Oriented Programming

Object oriented programming (OOP) focuses on using objects instead of using only logic and a sequence of instructions.

OOP is very important to avoid repetition, especially in large scale programs in all languages not only Python. By importing pre-existing modules from libraryâ€™s or creating your own module and classes all that is required is to program the instances or 'objects' that the program needs.

OOP Vocabulary

A class is a larger template or blueprint for code. After using the template to create the required code this is an instance of the class.

A class will contain the __init__ instance method which is the constructor called when the class is initialized. Functions inside of the class are instance methods and the variables in a class are instance variables.

OOP Inheritance

A sub class (child) can inherit attributes from a parent class. However a subclass can overwrite the parent class if needed. This is known as method overwriting.
