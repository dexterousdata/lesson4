Servers

A user requests a page through their web browser of choice. A Get request is sent to the IP / URL of the servers that host the requested webpage. The webpage is then returned to the user where it is rendered in their browser

Validating Inputs

Validating user inputs is very inportant for security. If escaping is not used then HTML can be entered into submition boxes creating a security threat. This also assits the user experiance by keeping the page apearing the same while escaping is appropriatly used.

HTML Templates

HTML Templates are used by programmers to avoid repotition alowing them to use their time more efficantly elsewere. By not having to retype code that can be stored in a template programmers save a lot of time.
