Lesson 2.3 - Procdures

Functions

Functions or 'procdures' are sub programs within the larger program. They act to complete a task for each given input. The top program can call on a sub program at any time and give inputs. The funciton then provides the output the top program uses.

Making a function is creating the sub program. The internal procdures for the function. Calling the function or using it is 'running' the sub program to provide an out put that is used by the program that called it.

Avoiding Repotition

Functions help programmers avoid repotion by preforming a sub task for each and every given input instead of the programmer rewriting the coad for each input.

Return Statment

The return statement tells the sub program what to return to the program that called it. The program that called the function then will use the provided output in its code.
